## 📋 Descripción

Repositorio con ejercicios prácticos de programación en C# para preparar entrevistas técnicas para posiciones **Junior Developer** o **Programmer**.

Cada ejercicio está diseñado para ser:
- ✅ **Simple y legible** - Código claro sin sobreingeniería
- ✅ **Auto-contenido** - Un archivo, un ejercicio
- ✅ **Listo para ejecutar** - Copiar, pegar y probar
- ✅ **Explicado paso a paso** - Comentarios y lógica clara

## 📚 Ejercicios Actuales

| # | Ejercicio | Descripción | Técnicas |
|---|-----------|-------------|----------|
| 1 | **Count Digits** | Contar dígitos de un número usando división entre 10 | Bucles, manejo de negativos |
| 2 | **Reverse Number** | Invertir un número (12345 → 54321) | Módulo, división, manejo de signo |
| 3 | **Find Second Largest** | Encontrar el segundo número más grande en un array | Ordenamiento, búsqueda, manejo de duplicados |
| 4 | **Most Repeated Character** | Encontrar el caracter que más se repite en una cadena | Diccionarios, conteo, búsqueda de máximo |
| 5 | **Capitalize First Letter** | Convertir primera letra de una palabra a mayúscula | Strings, char.ToUpper(), Substring |
